# DecodeLabs Internship - Project 1

## To-Do List Engine
This is a terminal-based program built during my DecodeLabs Industrial Training. It allows users to dynamically add tasks to a list and view them.

### Core Logic Used:
* Dynamic Python Lists (`[]`)
* Appending data (`.append()`)
* Print loops for data display
  
### Key Features of the Engine:
🔹 Interactive operational menu system (Codes 1-3)
🔹 Real-time data validation (prevents empty task submissions)
🔹 Dynamic record insertion logic 
🔹 RAM clearance & safe system shutdown engine
